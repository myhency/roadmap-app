## 초기 기능

- **대시보드**: 연도별 목표 진행 현황 (type/team/product별 통계)
- **간트차트**: Goal → Milestone → Task 계층 시각화
- **목표 관리**: Goal/Milestone/Task CRUD
- **멤버 관리**: 팀원 정보 관리
- **아이디어**: 제안 등록, 우선순위, 상태 관리, 댓글
