# API 변경 이력

API 엔드포인트 및 라우터 변경 사항

---

## 초기 구조

### 기본 엔드포인트

- `GET /` - 대시보드 페이지
- `GET /api/dashboard/summary` - 진행 요약
- `GET /api/years` - 연도 목록
- `GET /api/gantt/data` - 간트 데이터
- `/api/goals/` - Goal CRUD
- `/api/milestones/` - Milestone CRUD
- `/api/tasks/` - Task CRUD
- `/api/members/` - Member CRUD
- `/api/ideas/` - Idea CRUD
